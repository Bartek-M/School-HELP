# School Help
Here you will find tasks from school and explanations for given topics.

## Requirements
- Python 3.8 or above
- MinGW-w64 
- XAMPP 3.3.0 or above

## Languages
- Python
- C++
- PHP

## Running
Python
```bash
python <file_name>.py
```

C++
```bash
g++ -o run <file_name>.cpp
run.exe
```

PHP
```
Use XAMPP with Apache web server
Put PHP files in XAMPP/htdocs/ and open localhost
```
